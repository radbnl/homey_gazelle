Gazelle Token Generator

Genereer een token voor de Gazelle Connected Homey app.

Stap 1: Inloggen

Klik op de knop om in te loggen met je Gazelle account. Er opent een nieuw venster.

Let op: Na het inloggen kom je op een pagina die mogelijk een foutmelding toont. Dit is normaal! Kopieer de volledige URL uit je adresbalk.

Stap 2: Callback URL

Na het inloggen kom je op een pagina met een URL die begint met:
